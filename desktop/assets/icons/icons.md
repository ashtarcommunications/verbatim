## Convert an SVG
`convert-fa-icon.bat <name of svg without extension>`

e.g. for hourglass.svg, run:

`convert-fa-icon.bar hourglass`

## Font Awesome icon name mapping
* box-archive --> tub
* eye-low-vision --> eye
* hourglass --> hourglass
* minimize --> shrink
* share-nodes --> share
* table-columns --> arrange
* usb --> usb
